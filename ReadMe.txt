Parser_Brain(1.0) призначене для парсингу інформації по коду товару з оптового ресурсу Brain - з подальшим збереженням у форматі
.xlsx для імпорту в магазин на Prom.ua.

Інструкція з використання.

1. Для створення новогу файлу з позиціями - необхідно перемістити файл з кодами товарів (зразок заповнення буде наведений в
корневій папці проєкту під назвою "Example.xlsx") в папці проекту (Наприклад: С:\\programs\Parser_brain\create_new_file\file_download).
Після необхідно буде запустити скрипт Create.php, що в папці проєкту.
Почнеться обробка позицій очікуйте - в середньому час складатиме 2 секунди на 1 позицію. Файл створиться в папці С:\\programs\Parser_brain\create_new_file\
Щоб прервати операцію натисніть Ctrl + C .
ВАЖЛИВО: в папці .\file_download\ повинен знаходитись один .xlsx файл для обробки. Якщо потрібно згенерувати новий, видаліть
старий і перемістіть новий.

2. Скрипт Update.php - призначений для оновлення наявності і актуальної ціни по товарах. Перед запуском додайте файл
раніше створений за допомогою скрипта Create.php в папці проекту .\update_file\ . Запустіть скрипт Update.php і очкуйте
в середньому час складатиме 1 секунди на 1 позицію. Дані будуть оновлюватися в самому файлі, що був доданий.
ВАЖЛИВО: в папці .\update_file\ повинен знаходитись один .xlsx файл для обробки. Якщо потрібно оновити інший файл, видаліть
старий і перемістіть новий.

3. Важливо:
 1. Необхідно щоб було встановлено PHP(версія від 8.0) для запуска скрипта.
 2. Ціна товару  по коефіцієнту розраховується за формулою -  (опт_ціна + ((роздрібна_ціна - опт_ціна) * коефіцієнт)).
    Чим більше коефіцієнт, тим більша ціна 1 = роздрібну_ціна. Якщо пропущені позиції з коефіцінтом то середній автоматично
    встановлюється 0.5. Якщо хочете внести для всіх товарів поставити свої коефіцієнти, не пропускайте позиції і ЗАПОВНІТЬ ВСІ.
